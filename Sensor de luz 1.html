Funcionamiento de Sensor de Luz


Este sensor es el encargado de detectar la presencia o ausencia de luz,
si se detectan bajos niveles de luz se activara el servo motor y cerrara el toldo protector,
una vez se detecte la luz suficiente se activara el servo motor para retraer el toldo protector
