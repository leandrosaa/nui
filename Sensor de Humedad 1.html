Funcionamiento de Sensor de Humedad


Este sensor es el responsable de medir los niveles de humedad del suelo,
al detectar bajos niveles se activara un servo motor para abrir un grifo y liberar agua,
una vez se alcanza el nivel de humedad óptimo el servo motor cerrara el grifo.
