Funcionamiento de Sensor de Proximidad


Este sensor es el responsable de detectar la presencia de la fauna que puede afectar la integridad de las plantas,
en caso de detectar alguna, activara un servo motor cerrando el toldo protector,
una vez que el peligro para los cultivos se aleje el sensor activara denuevo el servo motor para retraer el toldo.
